# WarPrompts

> Juego de estrategia RTS 1v1 donde las unidades se controlan mediante prompts enviados a una IA (LLM).

🌐 **Dominio oficial:** [warpromptsgame.com](https://warpromptsgame.com)

## 🎮 Concepto

**WarPrompts** es un juego de estrategia en tiempo real con vista aérea donde dos jugadores compiten para conquistar un mapa. La mecánica central: las unidades se controlan a través de **prompts de lenguaje natural** enviados a un LLM. El jugador que mejor sepa comunicarse con la IA, mejor moverá su ejército.

## ⚡ Potenciadores

- 🔡 Prompt extendido — más caracteres para órdenes más complejas
- 🔍 Espionaje — ver el prompt del enemigo sin que lo sepa
- ⏳ Tiempo extra — más segundos para escribir
- 🔄 Reintento de IA — reinterpretar el prompt
